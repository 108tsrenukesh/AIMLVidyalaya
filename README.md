## 📲 Install it as an app (do this, seriously)

Once installed, Vidyalaya gets a home-screen icon, goes full-screen (bye-bye browser bars), loads faster, and **works offline** for lessons you've already opened. On Indian mobile networks, this matters.

### 🤖 Android (Chrome)

Tap the **Install** button in the app header, or hit Chrome's **⋮ menu → "Install app."** Done.

### 🍎 iPhone / iPad (Safari)

Apple being Apple — no automatic prompt. You gotta do it yourself:

1. Tap the **Share** icon (the box with the arrow pointing up).
2. Scroll down, tap **"Add to Home Screen."**
3. Tap **Add.** Done. It's an app now.

*(The in-app Install button on iOS shows you these same steps — it's not being lazy, it's being helpful.)*

### 💻 Desktop (Chrome / Edge)

Click the little **install icon** in the address bar, or use the in-app **Install** button.

> 📶 Once installed, lessons you've opened stay readable even on airplane mode or when your Wi-Fi decides to take a break.

**🌱 Complete beginner**

Read only the **Intuition / Visual Primer** of all three tracks first. Skip the math. Build the big picture. Then go deep when you're ready.

## 🤝 For contributors

Vidyalaya is **content-as-data**. Each lesson is a self-contained HTML file under `content/<track>/`. The app auto-discovers everything via a build script.

**To add a new lesson:**

1. Drop your `.html` file into `content/<track>/`.
2. Run `npm run build` (or push to `main` — GitHub Actions handles it).
3. That's it. The app picks it up automatically.

**Keep the three-layer pattern** (intuition → reference → projects) so every track stays consistent for learners.

**Local dev:**

```bash
npm install
npm run dev
```

Then open `http://localhost:5173/AIMLVidyalaya/`.

**Tech stack:**

React 19 · Vite · react-router-dom (HashRouter) · vite-plugin-pwa (Workbox) · GitHub Pages
