Field Card — RNNs at a glance

🧮 Core formulas

  • RNN ht=tanh(Wxxt+Whht-1+b)
  • LSTM Ct=f⊙Ct-1+i⊙g ; ht=o⊙tanh(Ct)
  • gates f,i,o=σ(W·[h,x]) ; g=tanh(Wg·[h,x])
  • GRU reset+update gates, no cell state
  • attn α=softmax(score); ctx=Σα·henc
  • PPL exp(mean cross-entropy)

🦓 Cell zoo — when to use

  • vanilla teaching only; forgets long range
  • GRU default: fast, data-efficient
  • LSTM long/complex deps; separate memory
  • BiRNN full sequence known (tagging) — not streaming
  • deep 2–4 stacked layers, rarely more

🕰️ Architecture timeline

  • 1990 Elman RNN — memory is born
  • 1997 LSTM — gates beat vanishing grads
  • 2014 GRU + seq2seq — encoder/decoder
  • 2014–15 attention — kills the bottleneck
  • 2017 Transformer — drop the loop
  • 2023+ Mamba / RWKV — recurrence returns

🔀 Sequence shapes

  • 1→many image → caption
  • many→1 review → sentiment
  • many→many tag each token (NER)
  • seq2seq translate, summarise, speech→text

Name the shape first — it picks the wiring.

🎯 Task → architecture

  • sentiment/intent BiLSTM/GRU → linear
  • NER/tagging BiLSTM + CRF
  • forecasting LSTM + multi-step head
  • translation seq2seq + attention
  • long-seq/edge SSM (Mamba) / small GRU
  • large-scale NLP Transformer / LLM

🏋️ Training essentials

  • clip grad-norm 1–5 — always (stops NaN)
  • teacher forcing feed true prev token
  • dropout variational (shared mask), not per-step
  • init orthogonal recurrent matrix
  • pack+mask ignore padding in loss
  • state carry+detach across long chunks

📏 Eval metrics

  • LM/gen perplexity (lower=better)
  • translate BLEU / ROUGE
  • classify accuracy / F1 (imbalance→F1)
  • forecast MAE / RMSE / MAPE + business cost

🎲 Decoding knobs

  • greedy fast, bland
  • beam translation (one right answer)
  • sample+temp creative; temp↑=wilder
  • top-k / top-p modern fluent default

🚀 Production & streaming

  • streaming RNN edge: keep state, O(1)/step
  • latency sequential — no across-time parallel
  • compress INT8 + prune → microcontrollers
  • export ONNX / TFLite / Core ML (mind state)
  • drift monitor error → retrain on schedule

🪤 Top traps

  • No grad clip → NaN
  • Global scaling/shuffle time → leakage
  • Unmasked padding → lying metrics
  • Per-step recurrent dropout → broken memory
  • Teacher-forcing only → exposure bias
  • BiRNN for streaming → impossible at inference

🧭 2026 defaults

  • small data / short seq → GRU/LSTM
  • streaming / edge / always-on → small quantised GRU
  • very long sequences on a budget → Mamba/SSM
  • large-scale language → Transformer / fine-tune LLM
  • forecasting → LSTM + business-cost metric, watch drift

🔍 Interpretability lenses

  • hidden traj PCA/t-SNE of states over time
  • gate audit plot forget-gate (1=keep,0=wipe)
  • grad flow log clip-norm: vanish↘ / explode↗
  • attn map alignment heatmap = free explanation

🧩 Specialized tools

  • CTC alignment-free loss → speech, OCR
  • LSTM AE reconstruct normal → anomaly score
  • Mamba/S6 selective SSM: Δ,B,C=f(x), O(L) scan
  • discretize A̅=exp(ΔA), zero-order hold
